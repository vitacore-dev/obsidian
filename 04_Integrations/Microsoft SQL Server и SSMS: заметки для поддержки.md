## Описание

Краткая справка по Microsoft SQL Server и SSMS для задач технической поддержки.

## Инструмент

- `SQL Server Management Studio (SSMS)` — GUI для выполнения запросов и работы с объектами БД.

## Типовые действия

- Выполнять диагностические запросы в SQL-редакторе.
- Проверять состояние сессий и процессов.
- Использовать runbook для завершения зависших сессий.
