Если при запуске клиента выходит ошибка:
```System.Exception: Ошибка старта компонента 'VCLib.HolderSettingsHolder'. System.Exception: Ошибка загрузки холдера.
 ---> VCLib.Common.ProcessException: Ошибка передачи параметров на сервер для выполнения метода "RunProcess"
 ---> System.Exception: Ошибка передачи параметров на сервер для выполнения метода "RunProcess"
 ---> System.Net.Http.HttpRequestException: An error occurred while sending the request.
 ---> System.IO.IOException: Unable to read data from the transport connection: Попытка установить соединение была безуспешной, т.к. от другого компьютера за требуемое время не получен нужный отклик, или было разорвано уже установленное соединение из-за неверного отклика уже подключенного компьютера..
 ---> System.Net.Sockets.SocketException (10060): Попытка установить соединение была безуспешной, т.к. от другого компьютера за требуемое время не получен нужный отклик, или было разорвано уже установленное соединение из-за неверного отклика уже подключенного компьютера.
```

Для устранения необходимо через cmd выполнить несколько шагов:
1. Проверить ping  к серверу  - если пинг есть, значит есть сеть;
2. Просмотр списка всех активных сетевых подключений, главным из которых является #MTU (отличается ли значение от 1300) с помощью команды: 
```
netsh interface ipv4 show subinterfaces
```
3. Далее изменить размер пакетов, чтобы улучшить стабильность соединения (команда на выставление #MTU 1300) с правами админа:  
```
netsh interface ipv4 set subinterface Ethernet mtu=1300 store=persistent
```
3. Перезапуск ПК и проверка запуска клиента
