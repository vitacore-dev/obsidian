1. Открываем осмотр через администратора МО![[Pasted image 20260525161128.png]]
2. Открыв осмотр копируем себе в notepa++ данные
![[Pasted image 20260525161252.png]]
3. На сущности medical_examination находим нужную запись, копируем идентификатор записи
![[Pasted image 20260525161422.png]]

4. Удаляем в БД запись по этому идентификатору
```
    delete [AKUZ].[f_medical_examination_examination]
  where id='e7913b94-d292-47c1-bec1-a5f809bf536d'
```
5. Открываем осмотр для редактирования как в п. 1 вставляем информацию при редактировании
