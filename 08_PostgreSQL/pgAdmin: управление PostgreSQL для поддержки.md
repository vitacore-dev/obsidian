## Описание

`pgAdmin` — GUI-инструмент для работы с PostgreSQL: выполнение запросов, просмотр объектов БД и базовые административные действия.

## Полезные действия

- Сброс интерфейса: `File -> Reset Layout`
- Увеличить масштаб интерфейса: `Ctrl++`
- Уменьшить масштаб интерфейса: `Ctrl+-`

## Когда использовать

- Быстро проверить состояние БД без работы в консоли.
- Выполнить диагностический SQL-запрос.
- Проверить объекты и настройки подключения.
